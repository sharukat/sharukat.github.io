Projects



Automatic ML Documentation Customization

Conducted an in-depth analysis of the existing machine learning documentation through a comprehensive case study focusing on TensorFlow documentation. Additionally, actively engaged in developing an automated tool to tailor documentation according to user expertise utilizing conventional and deep learning-based Natural Language Processing (NLP) techniques.

Near Duplicate Image Identifier

Designed an algorithm in Python for identifying duplicate text images such as invoices and medical claims, utilizing image processing techniques and the AKAZE descriptor detection method. The primary objective is to detect fraud and altered images effectively. Implemented the backend and corresponding APIs using Django and the Django REST framework.

ML-based Intrusion Detection

Conducted research to identify the most effective Machine Learning algorithm for Intrusion Detection Systems (IDS). Evaluated six ML algorithms (DNN, SVM, K-NN, One-Class SVM, K-Means, and EM) trained on NSL-KDD and UNSW-NB15 datasets. Implemented a robust real-time IDS, leveraging the power of a Deep Neural Network within a ML pipeline. The system ensures an effective approach to identifying potential security threats.

Password based Smart Door Lock

Executed the development of a smart door lock system programmed in the C programming language. The primary objective is to prevent unauthorized access and promptly alert the owner. This system is built upon an ATmega microcontroller and incorporates a GSM module for enhanced functionality.

Automated Car Parking System

Developed a comprehensive car park management system that visualizes available and occupied slots, facilitates reservations, generates receipts, and sends notifications for exceeded time limits. This was achieved through the creation of a desktop application programmed in Java, with MySQL utilized for efficient data management.

Dual Axis Solar Tracking System

Designed and implemented a solar tracking system capable of horizontal and vertical rotation to align the solar panel perpendicularly with sunlight. This system is powered by a PIC microcontroller programmed in C and integrates Light Dependent Resistors (LDRs) along with DC motors for precise positioning.
